# Backups Directory

This directory is for two things:
- frozen reference snapshots of historically strong configurations
- local safety backups made before risky edits

## Reference snapshots

Folders like `backups/best_20260202_28p0_500/` preserve code and notes from notable past runs. They are useful as comparison points, not as the active development path.

## Local backup archives

Ad hoc archives such as `backups/pre_github_cleanup_*.tgz` are local safety snapshots. They are ignored by git.
